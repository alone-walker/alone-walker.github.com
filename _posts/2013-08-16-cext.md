ext.c:  

{% highlight c %}
char arr[] = "Hello";
{% endhighlight c %}

main.c:  

{% highlight c %}
#include <stdio.h>  
#include <stdlib.h>  

extern char * arr;  

int  
main(int argc, char **argv) {  
    fprintf(stdout, "%.4s\n", (char *)&arr);  
    return EXIT_SUCCESS;  
}
{% endhighlight c %}

程序输出：Hell    
如果printf arr，会crash。正确的extern方法是：  

{% highlight c %}
extern char arr[];
{% endhighlight c %}
